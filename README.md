# FastAPI Projects

A collection of projects built with **FastAPI** — each in its own folder with its own README, dependencies, and setup instructions.
All projects share a common virtual environment located at `env/` in this root folder.

---

## Projects

### 1. [Simple Social](./Social_App/README.md)

A lightweight social media API and browser UI.

**Key features:**

- JWT-based user registration and login via `fastapi-users`
- Image and video post uploads stored on ImageKit CDN
- Public feed with author info and post ownership
- User profile management (bio, location, website, profile picture)

**Stack:** FastAPI · asyncpg · SQLAlchemy (async) · PostgreSQL on Supabase (PgBouncer pooler) · ImageKit · Streamlit frontend

**Folder:** `Social_App/`
